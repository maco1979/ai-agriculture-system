# 大宗交易分析系统 - 用户快速指南

## 系统简介

大宗交易分析系统是一个自动化工具，专门用于每日抓取和分析开盘半小时内的大宗交易信息。系统已配置为每日09:30自动执行，为投资者提供及时的市场洞察和决策参考。

## 快速开始

### 1. 系统状态检查
```bash
# 运行系统验证
python verify_system.py
```

### 2. 手动执行分析
```bash
# 运行完整分析流程
python run_automation.py
```

### 3. 查看生成文件
分析完成后，在 `d:\1.6\1.5` 目录中查看：
- `block_trading_report_YYYYMMDD.md` - 综合报告
- `opening_analysis_report_YYYYMMDD.md` - 专门分析
- `block_trading_data_YYYYMMDD.csv` - 原始数据
- `opening_analysis_chart_YYYYMMDD.png` - 可视化图表
- `automation_summary_YYYYMMDD.md` - 执行摘要

## 核心功能

### 1. 数据抓取
- 从多个数据源获取大宗交易数据
- 自动处理数据格式和编码
- 支持历史数据回补

### 2. 多维分析
| 分析维度 | 主要内容 | 投资应用 |
|----------|----------|----------|
| **时间模式** | 交易强度、大额交易特征 | 判断市场活跃度 |
| **行业板块** | 行业分布、热门板块 | 发现资金流向 |
| **价格模式** | 折溢率统计、价格区间 | 分析市场情绪 |
| **机构行为** | 买卖方营业部、集中度 | 跟踪机构动向 |

### 3. 报告输出
- **综合报告**: 完整的统计分析
- **专门报告**: 开盘半小时深度分析  
- **可视化图表**: 直观的数据展示
- **执行摘要**: 自动化执行状态

## 自动化配置

### 执行计划
- **执行时间**: 每日09:30（开盘后半小时）
- **执行频率**: 每个交易日自动执行
- **跳过规则**: 自动跳过周末和节假日
- **工作目录**: `d:\1.6\1.5`

### 自动化管理
- 配置位置: `.workbuddy/automations/automation/`
- 记忆文件: `memory.md` (记录执行历史)
- 状态监控: 通过执行摘要查看运行状态

## 常见问题

### Q1: 系统没有自动执行怎么办？
- 检查系统时间是否正确
- 验证自动化配置是否启用
- 查看 `.workbuddy/automations/automation/memory.md` 状态

### Q2: 数据获取失败怎么办？
- 检查网络连接是否正常
- 验证数据源API是否可用
- 查看错误日志中的详细提示

### Q3: 报告内容不完整怎么办？
- 检查输入数据质量
- 验证依赖库版本是否匹配
- 查看系统验证结果

### Q4: 如何修改分析参数？
- 编辑 `opening_analysis.py` 中的相关参数
- 调整行业分类规则、大额交易阈值等
- 修改后运行 `python verify_system.py` 验证
