## 🚀 部署步骤

### 步骤1：准备代码仓库

1. **创建GitHub仓库**
   ```
   仓库名：ai-agri-system
   描述：AI农业决策系统 - 零成本SaaS
   公开仓库
   ```

2. **上传代码到GitHub**
   ```bash
   # 在项目目录执行
   git init
   git add .
   git commit -m "初始提交：AI农业决策系统"
   git branch -M main
   git remote add origin https://github.com/你的用户名/ai-agri-system.git
   git push -u origin main
   ```

### 步骤2：部署前端到Vercel

1. **登录Vercel** (使用GitHub账号)
2. **导入项目**
   - 选择GitHub仓库：ai-agri-system
   - 选择根目录：frontend/
   - 框架预设：Vite
   - 构建命令：npm run build
   - 输出目录：dist

3. **环境变量配置**
   ```
   VITE_API_URL=https://你的后端地址.railway.app
   ```

4. **自动部署**
   - Vercel会自动部署并生成域名
   - 免费域名：ai-agri-system.vercel.app

### 步骤3：设置Supabase数据库

1. **创建Supabase项目**
   - 项目名：ai-agri-db
   - 数据库密码：设置强密码
   - 地区：选择最近地区

2. **获取连接信息**
   - 项目URL：https://你的项目.supabase.co
   - API密钥：anon public key
   - 数据库连接字符串

3. **导入数据** (可选)
   - 使用Supabase SQL编辑器
   - 导入现有数据库结构

### 步骤4：部署后端到Railway

1. **创建Railway项目**
   - 项目名：ai-agri-backend
   - 从GitHub部署

2. **配置服务**
   - 选择目录：backend/
   - 构建命令：pip install -r requirements.txt
   - 启动命令：python main.py

3. **环境变量配置**
   ```
   DATABASE_URL=postgresql://supabase连接字符串
   REDIS_URL=可选，暂时不用
   API_KEY=你的API密钥
   ENVIRONMENT=production
   ```

4. **获取API地址**
   - Railway会自动分配域名
   - API地址：https://ai-agri-backend.up.railway.app

### 步骤5：配置Cloudflare

1. **添加域名** (如果需要自定义域名)
   - 添加你的域名到Cloudflare
   - 配置DNS指向Vercel

2. **设置SSL**
   - Cloudflare自动提供免费SSL
   - 开启Always Use HTTPS

3. **配置CDN缓存**
   - 缓存静态资源
   - 优化图片加载

### 步骤6：设置监控

1. **Better Stack监控**
   - 添加前端监控：Vercel域名
   - 添加后端监控：Railway API
   - 设置告警：响应时间>2秒

2. **错误跟踪**
   - 配置错误报告
   - 设置性能监控

## 🔧 配置调整

### 前端配置 (frontend/)
1. **修改API地址**
   ```javascript
   // 在环境配置中
   const API_BASE_URL = import.meta.env.VITE_API_URL || 'https://ai-agri-backend.up.railway.app';
   ```

2. **构建优化**
   ```json
   // package.json
   {
     "scripts": {
       "build": "vite build",
       "preview": "vite preview"
     }
   }
   ```

### 后端配置 (backend/)
1. **数据库连接**
   ```python
   # .env 或环境变量
   DATABASE_URL=postgresql://username:password@host:port/database
   ```

2. **CORS配置**
   ```python
   # 允许Vercel域名访问
   origins = [
       "https://ai-agri-system.vercel.app",
       "http://localhost:3000"
   ]
   ```
